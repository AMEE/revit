# AMEE-in-Revit

A prototype Autodesk Revit Architecture 2012 addin that uses AMEEconnect to show 
environmental data for materials during design.

## INSTALLATION

No installation package is currently supplied, but you can build the code
using Visual C#.

## REQUIREMENTS

 * Autodesk Revit 2012
